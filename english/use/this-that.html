This / That का प्रयोग
( Use of This and That in hindi )

This- यह
That -वह

This का प्रयोग पास की वस्तुओं की तरफ इसारा करने के लिये किया जाता है |
That का प्रयोग दूर की वस्तुओं की तरफ इसारा करने के लिये किया जाता है |

This और That का प्रयोग केवल एक वस्तु की तरफ इसारा करने के लिये किया जाता है |
एक से ज्यादा पास की वस्तुओं लिये These
दूर की एक से अधिक वस्तुओं की के लिए Those का प्रयोग किया जाता है

Rule

प्रयोग करने के नियम -

  1. पास की वस्तु के लिए This
  3. दूर की वस्तु के लिए That
This / That + is + object .
Example
  1. यह राम है
    This is Ram
  2. वह किताब है
    That is book.

Negative

is के बाद Not लगातें हैं |

This/ That + is + not + object .
Example
  1. यह राम नहीं है |
    This is not Ram.
  2. वह किताब नहीं है
    That is not book.

Interrogative

ऐसे वाक्यों में प्रश्नवाचक बनाने के लिए is को सबसे पहले लगतें हैं

is + This/ That + object ?
Example
  1. क्या यह राम है|
    Is this Ram
  2. क्या वह किताब है|
    Is that book.
