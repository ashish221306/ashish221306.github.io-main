SEO META TAGS

SEO meta tag are very important for seo of any website to rank fast on google

In SEO (Search Engine Optimization) meta tags still play an important role. Especially the use of the title tag and meta description is very important.

  1. <title></title>

    Title tag tells the search engine whats is the page about

  2. <meta name="description" content="your description">

    Description tag tells the search engine what is the page about in details

  3. <meta name="keywords" content="your keywords">

    Keywords meta tag tells the search engine on which keyword you want to make focus for this page

  4. <meta name="language" content="your webpage's language">

    You use this meta tag to tell the spiders what the main language is that is used on the site.

  5. <meta name="author" content="author of webpage">

  6. <link rel="canonical" href="author of webpage">

    By implementing the Canonical tag in the code, we tell the search engines that this URL is the main page and avoid indexing other duplicate page URLs.

  7. <meta http-equiv="Cache-Control" content="no-store" />

    Google also uses the Cache feature. In stead of analyzing a webpage real-time, pages are stored in the database of a search engine. This is a method to generate search results more quickly.

    1. Public – may be cached in public shared caches
    2. Private – may only be cached in private cache
    3. no-Cache – may not be cached
    4. no-Store – may be cached but not archived
