jquery traversal of element

Learn how to traverse dom element by jquery
Traversal Methods do not modify the jQuery object and they are used to filter out elements from a document based on given conditions.

How to Find Elements by Index

  1. Every list has its own index, and can be located directly by using eq(index) method
  2. Every child element starts its index from zero, thus, list item 2 would be accessed by using $("li").eq(1) and so on.

Example

we can select the list element index as follows

Example

    <script type = "text/javascript" language = "javascript">
        $(document).ready(function() {
           $("li").eq(2).addClass("selected");
        });
     </script>

How to filter element

The filter( selector ) method can be used to filter out all elements from the set of matched elements that do not match the specified selector(s).

$("li").filter(".middle").addClass("selected");

Finding Element

The find( selector ) method can be used to locate all the descendant elements of a particular type of elements.

$("p").find("span").addClass("selected");

above example will find all span element under <p> tag and will add class selected

Jquery Dom Filter method

Following are jquery DOM filter methods

Sr.No. Method & Description
1 eq( index )

Reduce the set of matched elements to a single element.
2 filter( selector )

Removes all elements from the set of matched elements that do not match the specified selector(s).
3 filter( fn )

Removes all elements from the set of matched elements that do not match the specified function.
4 is( selector )

Checks the current selection against an expression and returns true, if at least one element of the selection fits the given selector.
5 map( callback )

Translate a set of elements in the jQuery object into another set of values in a jQuery array (which may, or may not contain elements).
6 not( selector )

Removes elements matching the specified selector from the set of matched elements.
7 slice( start, [end] )

Selects a subset of the matched elements.

Get the direct parent of an element. If called on a set of elements, parent returns a set of their unique direct parent elements.
