Javascript Timing Function

The JS window object provides two build in timing functions

  1. setInterval
  2. setTimeout

This is to repeat a function after certain time duration or to run a function once after certain time. Even HTML5 introduced a new timing function requestAnimationFrame to run smoother animations, mainly used in HTML5 Canvas.

setInterval

The Javascript setInterval method repeat a function after given time. The first parameter of setInterval is function name and second parameter is time (in milliseconds). As setInterval is a JS build in function, so no need to invoke function name in setInterval.
See example

    <script>

    function myAlert(){
      alert("my alert");
    }                  
    setInterval(myAlert,1000)  
    // repeat function after every 1 second                     
   </script>

How to make counter using timing function

    <button id="counter">start</button>
    <p id="pp"></p>
    <script>
    
    document.querySelector('#counter').onclick=function(){
        var counter=0;
    function increaseCounter(){
    counter++;
    document.querySelector("#pp").innerHTML=counter;
    }
    setInterval(increaseCounter,1000);
    }
    </script>

setTimeout

setTimeout method is used to call a function once but after a delay of some seconds. The first parameter of setTimeout method is function name and second is time in milliseconds. See example

    function msg()
    {
     alert("hey user");
    }             

    setTimeout(msg,1000)

      // run function after every 1 second

Example

clearInterval

clearInterval function is used to stop setInterval and setTimeout functions for further executions. The parameter in clearTimeout function is either id of setInterval/setTimeout or variable name of setInterval/setTimeout functions. 

   var counter=0;
   function setTimmer(){
   console.log(counter);
   counter++;
   }
  var t1=setInterval(setTimmer,1000);   
  
    clearInterval(t1);     // stop setTimmer function
