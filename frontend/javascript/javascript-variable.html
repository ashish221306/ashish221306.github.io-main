Javascript variable

JavaScript variables are containers for storing data values.

Example 
var x = 5;
var y = 6;
var z = x + y;

From the example above, you can expect:

  1. x stores the value 5
  2. y stores the value 6
  3. z stores the value 11

Variables are always initialized first with var keyword.
For exp var x means a variable named x is initialized.
To assign a value in variable, = or assignment operator is used, followed by value.

  1. variables names cannot include javascript reserved keywords, like var, for, in, while, true, false, null, class, let, const, function etc.
  2. variables cannot starts with numbers and hyphen, use alphabets or underscore(_).
  3. variables can have strings followed by numbers, like x1 and x2 are allowed but 1x, 2x is not allowed.
  4. For separation in variable names, use underscore (_) or camel casing, do not use hyphen (-) separation and white space, i.e, user_name is valid, username is also valid, but user-name is invalid.
  5. variables names are case sensitive, i.e, x1 and X1 are different.
