JS Window Object

JS Window Object is the global object in JavaScript. For a web browser, window object tell information about current browser window. Window Object is also the global object of browser environment.

window object is available in browser environment only. In other environment's, there is no window object. For exp, in Node JS, global is used instead of window object.

All global variables and functions are windows properties and methods

    var x="hello javascript";
    var y=function(){ return "hi how are you"};
    
    window.x;          // return "hello javascript"
    window.y();        // return "hi how are you"

Global const and let are not windows properties and methods.

    let x="hello js";
    let y=function(){ return "hi"};
    const pi=Math.PI;
    
    
    window.x;            // return undefined
    window.y;            // return window.y is not a function 
    window.pi            // return undefined

Window Methods

JS Window Methods are build in functions.

Method use
window.open() Opens a new browser window.
window.open("print.html","_blank","width=400,height=500,top=100,left=300,resizable=yes")
window.close() Close browser window opened by window.open method.
                window.close()
window.print() Prints the current browser window.
                window.print()
window.resizeTo() resize browser to parameter inside. 
window.resizeTo(600,400)
will resize window to 600 by 400.
window.moveTo move window to left top corner of screen
