Javascript Date Object

JavaScript Date Object is used to get date and time of user device. Using JS date object, we can detect date, time, timezone, day and also change date time data using js date function. JS Date Object is a build in constructor object is javascript.


JS will pick date and time from user machine. The accuracy of date and time totally depends upon user device.

new Date( )

new Date( ) functon is used to get date and time in javascript. Date is a constructor function using new operator.

    var date=new Date();            // new Date Object

new Date from EPOCH

    var date=new Date(0);            
    // new Date Object from EPOCH Date and Time

new Date function with date and time

    var date=new Date('2020-01-30T14:00');            
    // new Date Object with custom date and time

new Date function with date, time and timezone

    var date=new Date('2020-01-30T12:00:00Z');            
    // new Date Object with custom date, time & timezone
       // Z stands for Greenwich Mean Time

If system date is incorrect, date function will respond incorrect date and time.

getMilliseconds()

The getMilliseconds() method of date object returns millisecond form 0 to 999.

        var t=new Date();

        document.querySelector("p").innerHTML=msec;
