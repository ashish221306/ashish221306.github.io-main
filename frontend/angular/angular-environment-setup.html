Angular Environment Setup

Environment Setup required for angular

  1. Node js
  2. NPM
  3. Angular CLI
  4. IDE For writing your code

Nodejs is javascript runtime environment to run javascript out of browser. Node js should be greater than version 4 and NPM should be greater than version 3

Node js

To check if node js is installed or not on your system. type node -v in your prompt or your IDE terminal. This will show you the version of node js currently installed on your system / laptop. node -v if it does not show anything it means node js is not installed on your system.You will have to download Node js.aboutmedetails download Node js
or you can download from Node.JS official website https://nodejs.org/en/download/

NPM

NPM is node package manager which is used to download angular packeges in our project. type node -v in your prompt or your IDE terminal. This will show you the version of npm currently installed in your node js. npm -v it is installed with Node js which means that when you download Node.js , you automatically get npm installed on your computer. to update npm you can run this command. npm install npm@latest -g

Angular CLI

Angular CLI is used to run angular commands. To download angular CLI run following command in node js command line or command prompt but Node js should be already downloaded. > npm install -g @angular/cli

now your system is ready to create first angular project.
